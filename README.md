# 📊 EdTech Analysis Dashboard using Power BI

 **An interactive Power BI dashboard that analyzes online course data to uncover learner behavior, course performance, and business insights for an EdTech startup.**


# 📌 Project Overview

The **EdTech Analysis Dashboard** is a Business Intelligence project developed using **Power BI**. The project transforms raw online course data into meaningful insights that help an EdTech company understand learner preferences, evaluate instructor performance, optimize course offerings, and support strategic business decisions.

The dashboard answers key business questions through interactive visualizations, making it easier for stakeholders to identify growth opportunities and improve learner engagement.


# 🎯 Project Objective

The main objective of this project is to analyze online course data and provide data-driven insights that can help an EdTech startup:

* Improve learner engagement
* Optimize course offerings
* Identify high-performing categories
* Understand language preferences
* Enhance accessibility using subtitles
* Support business decision-making

# 🔄 Project Workflow

### 1️⃣ Data Collection

Collected the online course dataset containing information about course categories, instructors, languages, ratings, registrations, viewers, subtitles, and duration.

### 2️⃣ Data Cleaning

* Removed duplicate records
* Handled missing values
* Corrected inconsistent formats
* Standardized data types

### 3️⃣ Data Transformation

Performed data transformation using **Power Query** and prepared the dataset for analysis.

### 4️⃣ Data Modeling

Created relationships and calculated measures using **DAX** for KPIs and analytical metrics.

### 5️⃣ Dashboard Development

Designed interactive dashboards using:

* KPI Cards
* Bar Charts
* Pie Charts
* Tables
* Slicers

### 6️⃣ Business Analysis

Generated insights and recommendations based on dashboard findings.


# 📈 Key Insights

* More than **3,000** online courses were analyzed.
* The average course rating is approximately **4.7**.
* English is the dominant language across the platform.
* Data Science courses attract the highest learner engagement.
* Courses with subtitle support generally receive higher registrations.
* Business and Data Science categories have the strongest instructor performance.
* Course popularity varies significantly across different categories.


# 💡 Business Recommendations

* Invest more in high-performing categories.
* Increase multilingual course offerings.
* Provide subtitles for better accessibility.
* Collaborate with top-rated instructors.
* Focus on learner engagement metrics for future course development.
* Monitor category trends regularly for better decision-making.
